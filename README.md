## Project Description
Masar is an iOS service booking application that connects service seekers with service providers across multiple categories such as IT solutions, teaching, and digital services.  
The app supports booking management, communication, reviews, provider verification, and admin moderation.

## Project Setup Instructions
1. Clone the repository.
2. Open the project using **Xcode 16.4**.
3. Ensure `GoogleService-Info.plist` is included.
4. Install dependencies via **Swift Package Manager**.
5. Select a simulator and run the project.
